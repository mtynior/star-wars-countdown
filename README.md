## Build Setup

```bash
# Project setup
yarn install

# Compiles and hot-reloads for development
yarn run serve

# Compiles and minifies for production
yarn run build

# Lints and fixes files
yarn run lint

# Run tests
yarn run  test

# Run unit tests
yarn run test:unit
```
